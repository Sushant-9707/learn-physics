Scalars & Vectors

Physical quantities describe the world, but not all quantities behave the same. Some require only magnitude, while others demand both magnitude and direction.

Core Ideas

Scalar Quantities

Scalar quantities are described completely by magnitude alone. Examples include mass, time, temperature, energy, and speed.

Scalars can be added or subtracted using ordinary algebra without considering direction.

Vector Quantities

Vector quantities require both magnitude and direction for complete description. Examples include displacement, velocity, acceleration, and force.

Changing the direction of a vector changes the physical meaning, even if magnitude remains the same.

Representation of Vectors

Vectors are commonly represented by arrows. The length of the arrow represents magnitude, while the arrowhead indicates direction.

Vectors can also be represented using components along coordinate axes.

Addition of Vectors

Vectors are added using geometric methods such as the head-to-tail method or parallelogram law.

Algebraic addition using components is often more efficient for calculations.
