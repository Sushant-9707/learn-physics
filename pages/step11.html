    Rotational Kinematics

    Rotational kinematics describes how objects rotate. Instead of position and velocity in a straight line, we now deal with angles, angular velocity, and angular acceleration.

    Core Ideas

    Angular Variables

    Rotational motion is described using angular displacement instead of linear displacement. Angles measure how much an object has rotated about an axis.

    These angular quantities are independent of the object's size.

    Angular Velocity

    Angular velocity describes how fast an object rotates. It measures the rate of change of angular displacement.

    All points on a rigid body have the same angular velocity, even though their linear speeds may differ.

    Angular Acceleration

    Angular acceleration measures how quickly angular velocity changes with time.

    It plays the same role in rotational motion as linear acceleration does in straight-line motion.

    Equations of Rotational Motion

    When angular acceleration is constant, rotational motion follows equations similar to linear kinematics.

    These equations allow prediction of angular position and velocity over time.
